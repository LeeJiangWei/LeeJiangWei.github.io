遍历

显然链表结果是前序遍历结果，先遍历，然后连成链表即可。

递归

拉成链表可以视为以下 3 步：

  1. 将 root 的左子树和右子树拉平。
  2. 将 root 的右子树接到左子树下方
  3. 然后将整个左子树作为右子树。
